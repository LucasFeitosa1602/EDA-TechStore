# 📊 EDA - Tech Store Sales Analysis

## 📌 Sobre o Projeto
Este projeto realiza uma Análise Exploratória de Dados (EDA) em um dataset de vendas de uma loja de tecnologia (Tech Store), com o objetivo de extrair insights sobre desempenho de vendas, comportamento por categoria e distribuição regional.

A análise foi desenvolvida com Python, focando em transformação de dados, visualização e geração de métricas relevantes para negócio.

---

## 🧰 Tecnologias Utilizadas

- Python  
- Pandas  
- NumPy  
- Matplotlib  
- Seaborn  
- Jupyter Notebook  

---

---

## 🔍 Etapas da Análise

### 📥 Carregamento dos Dados
- Leitura do dataset com Pandas  
- Verificação de estrutura e tipos de dados  

### 🧮 Feature Engineering
  - Criação da métrica:
  - Receita_Total = quantidade × preço  

### 📊 Análises Realizadas

- Receita por categoria  
- Produto mais vendido  
- Evolução mensal das vendas  
- Análise por região  

### 📈 Visualizações

- Gráfico de barras (receita por categoria)  
- Gráfico de linha (evolução mensal)  

### 📌 Tabela Dinâmica

- Receita por Região × Categoria  
---

## 📊 Insights

- Identificação das categorias mais lucrativas  
- Regiões com maior volume de vendas  
- Tendência de crescimento ao longo do tempo  
- Produtos com melhor desempenho
